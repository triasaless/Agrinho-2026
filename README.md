# 🌽 BioAgro - Do Milho ao Plástico (e à Música!)

Um simulador interativo web que une tecnologia, agronomia sustentável e economia circular. O projeto demonstra como a produção agrícola pode ir além da alimentação, transformando matéria-prima renovável em subprodutos tecnológicos de alto valor agregado.

## 🚀 Como funciona?
O usuário insere a quantidade de milho colhido (em toneladas) e o sistema calcula em tempo real:
1. A estimativa de **bioplástico (PLA)** biodegradável que pode ser gerado.
2. O impacto cultural e sustentável, convertendo esse plástico na quantidade equivalente de **discos de vinil ecológicos** 🎶.

## 🛠️ Tecnologias Utilizadas
O projeto foi construído do zero, utilizando as tecnologias fundamentais do desenvolvimento web para garantir leveza e performance:
* **HTML5:** Estruturação semântica da página.
* **CSS3:** Design responsivo, moderno e estilizado com foco na temática agro-sustentável (layouts limpos e transições suaves).
* **JavaScript (ES6):** Lógica de programação para manipulação do DOM, validação de dados e cálculos dinâmicos sem recarregamento de página.
